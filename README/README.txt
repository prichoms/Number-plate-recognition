STEPS TO RUN THE PROJECT
-------------------------

The project contains four folders - Dataset, Code, Readme and Screenshots. 
You need to unzip and extract the folder and run the ipynb file in the code subfolder.
The dataset folder ought to be uploaded in your google drive under 'my drive' section, in order to be called from the colab command.
All the instructions and directions needed to run the code are already provided in the ipynb file.
